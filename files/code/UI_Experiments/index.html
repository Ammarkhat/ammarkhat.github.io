WebGl Experiments

 Some WebGl experiments that I wrote during my PhD research:

  •   Generate Rough Cutting Instructions (Unpublished yet)
    The goal of this experiment is compute rough cutting instructions to help the user in roughly cutting any 3D shape.
    [Try it online] *note: press ENTER.
  •   Compute Mesh Visibility
    The goal of this experiment is to measure how much visible each point in the mesh (blue: more visible, red: less visible).
    [Try it online] *note: press ENTER.
  •   Deformable Changes Synchronization
    In the project of Differential 3D Scanning, when the physical changes are smooth (As captured in the scanned point cloud). We propose a deformable model, where we deform the reference model until it matches the point cloud.
    [Try it online]*note: click DeformMesh button to start the deformation.
  •   3D Laser Scanning Simulator
    To understand the 3D scanning process, and to find the best structure for the 3D scanner, we built a 3D laser scanning simulator; where the user can upload his own 3D model and then set the position and orientation for a number of laser lines, and the camera, then he can start the simulation. The result point cloud will try to approximate the real 3D scanning result.
    [Try it online]
  •   Inflate Meshes
    The goal of this experiment is to inflate a mesh by expanding it in the areas of negative curventures.
    [Try it online] *note: press ENTER.
  •   Medial Axis Points
    The goal of this experiment is to compute some points on the medial axis using the method of NCH.
    [Try it online] *note: press ENTER.
  •   CAD Mesh Segmentation
    The goal of this experiment is to segment a mesh using mesh faces normals vectors.
    [Try it online] *note: press ENTER.
  •   Compute Sharp Edges
    The goal of this experiment is to compute all sharp edges in a mesh.
    [Try it online] *note: press ENTER.
  •   Mesh Coloring
    A fun way to color a 3D mesh.
    [Try it online] *note: press ENTER.
